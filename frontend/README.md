# Freeweight Mobile App

React Native (Expo) frontend for the Freeweight strength training platform.

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

Then press:
- `i` for iOS simulator
- `a` for Android emulator
- `w` for web

## Environment

Backend API: `http://localhost:8001`

## Structure

- `src/api/` - API client and endpoint definitions
- `src/screens/` - Screen components organized by user type
- `src/navigation/` - React Navigation setup
- `src/hooks/` - Custom hooks including auth context
